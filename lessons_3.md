#MCP

##1: Context7
**Ссылка:** https://github.com/upstash/context7
**Что делает:** Дает доступ к актуальной документации библиотек.
**Почему выбрал:** Версии библиотек растут быстрее чем AI дообучаются.
В результате агенты генерят код на устаревших примерах, с устаревшими версиями в toml.
Context7 берет документацию прямо из исходников и вставляет в промпт.

##2: Claude in mobile
**Ссылка:** https://github.com/AlexGladkov/claude-in-mobile
**Что делает:** Дает агенту возможность отлаживать приложение через adb
**Почему выбрал:** Можно делегировать UI и интеграционные тесты агенту

##3: DeepWiki
**Ссылка:** https://docs.devin.ai/work-with-devin/deepwiki-mcp
**Что делает:** Собирает документацию по любому открытому репозиторию
**Почему выбрал:** Увеличивает скорость изучения новых библиотек

#SKILL
##1: Superpowers
**Ссылка:** https://github.com/obra/superpowers
**Что делает:** Набор скиллов, который выстраивает устойчивый процесс разработки
**Почему выбрал:** Лучшие практики для разработки фичей любой сложности

##2: RTK
**Ссылка:** https://github.com/rtk-ai/rtk
**Что делает:** Сжимает команды, отправляемые в LLM
**Почему выбрал:** Экономия токенов.

##3: AST Indext
**Ссылка:** https://github.com/defendend/Claude-ast-index-search
**Что делает:** Быстрый поиск по проекту для агента (индексирование)
**Почему выбрал:** Экономия токенов. Увеличение скорости сканирования проекта
